# DiffusionGemma Latency UX Lab

Diffusion language models may matter less as benchmark winners and more as a UX primitive for instant edits, structured repair, and low-latency local assistants.

Inspired by: https://x.com/fofrAI/status/2064749295165669581

## What it contains

- `artifact-package/` — Obsidian package note mirrored from x-intel.
- `prototype/` — concrete trial packet / checklist.
- `infographic/` — workflow explainer when available.
- `skill-draft/` — reusable procedure draft when available.
- `improvement-loop/` — recurring improvement tracker.

## Quickstart

Open the prototype markdown file and fill it during a real Colab/local latency trial, then fill `prototype/Latency Evidence Attachment Card.md` with command logs, per-task latency, qualitative UX notes, baseline comparison, and failure examples. Complete `prototype/Post-Trial Promotion Decision Card.md` only after the evidence bundle exists. This repo intentionally ships templates and gates, not fabricated validation results.
